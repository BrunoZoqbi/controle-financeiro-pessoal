Controle Financeiro Pessoal

Sobre o Projeto

Este projeto tem como objetivo estruturar um modelo simples e organizado de controle financeiro pessoal, permitindo melhor visualização de receitas, despesas e saldo ao longo do tempo.

A solução foi desenvolvida como parte da construção de portfólio profissional voltado para organização de processos, análise de dados e tomada de decisão baseada em informações estruturadas.

O projeto parte de uma necessidade prática: organizar finanças pessoais de forma clara, escalável e evolutiva.


Objetivo

Criar uma estrutura inicial de controle financeiro que permita:
  Registrar receitas e despesas
  Categorizar movimentações financeiras
  Acompanhar saldo mensal
  Melhorar a organização financeira
  Servir como base para evolução futura com análise de dados
  
Além do uso pessoal, o projeto também pode ser adaptado para:
  Pequenos negócios
  Gestão administrativa
  Controle financeiro básico organizacional
